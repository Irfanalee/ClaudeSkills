# Claude Skills

A collection of custom skills for Claude Code that enhance its capabilities across various domains. New skills are added regularly to expand functionality.

## What are Claude Skills?

Claude Skills are custom instruction sets that guide Claude Code to perform specialized tasks with expert-level precision. Each skill is defined in a markdown file with specific triggers, workflows, and domain knowledge.

## Skills Catalog

### DevSkills

| Skill | Description |
|-------|-------------|
| [CyberSec](DevSkills/CyberSec.md) | Security vulnerability scanner for code repositories - detects secrets, CVEs, SAST issues, and misconfigurations |

## Usage

To use these skills with Claude Code, add them to your Claude Code skills configuration or reference them in your project's `.claude/` directory.
